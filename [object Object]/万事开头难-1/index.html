万事开头难-1

发表于 2023-10-29 分类于 前言

写在干正事儿之前：

回忆一下自己是如何沦落到如此这般境地的。

冒险开始

一个各方面都很普通的小镇青年(我)用普通的成绩考上一个普通城市的一所普通大学，稀里糊涂地选择了一个自己都不知道是啥的专业：机械设计制造及其自动化。

我的学生生涯一直如此，在班里成绩中等偏上。父母从没有管过我学业，对我的要求也不高，能考个本科就行。那几年，这个小镇最顶尖的高中里有不到一半的人能考上本科。

大学我的成绩依然中等偏上。全专业100多号人，我的排名大概在前百分之20%以内。整体而言，我对自己大学学业的评价是中规中矩，不好，也不差。

专业表现方面，总结如下：

  1. 专业课和重要通识课（高数、大物等）少有逃课，上课经常看手机，作业按时交（抄也是交了）
  2. 无挂科，GPA3.5，无作弊（不是没有抓到的那种，是真没作弊）
  3. 独立完成课程设计和毕业设计，课设/毕设成绩85~90

相信大多数机械相关专业的人按上述三点来考量，应该跟我自评相差不远。

大四找工作，面试了三家企业：

  1. 第一家校招，佛山电机研发，技术岗，发完offer签完三方三个月后HR跟我说公司在打官司，说我要去的话也可以，但劝我看看有没有更好的选择。当时年轻不知道社会险恶，应该是他们找到更合适的或不想招人又不想赔钱的托词，想让我自己解除三方。我解除了，没有人会想去一家不欢迎自己的公司。
  2. 第二家校招，珠海做空调的，研发岗，面试和主管啥的都聊的很好，最后一面HR说技术主管觉得很不错，邀请我再来公司参观一下，顺便再见见日资方的负责人，没问题就发offer。后面因为第三家，拒绝了没去。
  3. 第三家社招，研发岗，虽然规模远远比不过第二家，但行业和Base都是我看好的，而且可以立马实习，果断选择，冒险继续。

以现在的眼光审视我的大学四年学业，除了会用画图软件（CAD/UG），其他专业方面是一点不会、一窍不通、一问三不知。当然，严格来讲：画图软件也只学到了皮毛。

冒险继续，但开头难

第一份工作很痛苦。

痛苦，一是工资低，实习工资不到三千块钱，吃住就花光，有时不够还得用花呗，因为经常要回学校对称毕业设计进度，车费也是不小的花销。
二是因为干着吃力，工作前觉得自己也不差：相比较于不少同学花钱买毕设、挂科、作弊等等行为，我觉得我是可以胜任这份工作的。但实际工作中发现不是这样，老师教的、书本上讲的不能说和工作毫无瓜葛，只能说理论和实践之间存比较大的偏差。而且，一边工作的同时要完成毕业设计，几乎没有娱乐时间。

对于信息透明度不高的行业（后来逐渐发现，在中文圈确是这样），经验很重要（经验在全世界都重要），别人为啥传授你经验？而且，中文互联网搜索到的相关信息良莠不齐。很多时候，百度检索到的信息是无效或失效的，这一点在越往后的工作中感受越深。

没办法，百度不到的就试着问同事。同事氛围整体上是不错的，但也被人嘲讽过“你怎么连这都不会？”，从此不敢再问别人。只能先做事，做出来不行再改，一个件一天能改数个版本，改来改去自己都不知道在改啥了。

更痛苦的是转正后，我开始独自负责某一子系统。独自负责不是因为我优秀，而是负责这块的同事要离职了，没招到合适的人就让我来顶班。同事离职前带我熟悉了这一块，知道了大概是咋回事，这点还是感谢，不然我早已提桶跑路。

独自负责的子系统是一个涉及多学科交叉的系统。需要具备一定的机械、流体、软件硬件甚至算法方向（当然更少不了数学和物理方面，这是基础；还有其它跟机械工程不相干的专业）的专业知识。在这个过程中跟各路牛鬼蛇神打交道，心力交瘁，一方面是面对没有接触过的专业领域感到恐惧和迷茫，另一方面是对公司的人和事感到失望——而且是随着对自己专业工作的深入了解，变得越来越失望。

转正后干了半年，离职，去另一家公司干类似的工作。

算上实习半年，在这家公司待了1年。实习半年和转正半年干的本质上不是一类工作。实习期学会的东西对我后面的工作是有益的，不过转正后的工作可以说是从零开始。

虽然被同事离职影响，从而偏离了传统机械工程师的工作方向，但我还是跌跌撞撞干到了现在。
